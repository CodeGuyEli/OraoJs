QUICKSTART

  1. Type in BC and press ENTER
  2. Press ENTER when asked about mem size
  3. Type LMEM "DEMO2" (with quotes, shift+2 to get them)
  4. If you screw up typing, left arrow to the rescue
  5. You can also try: LMEM "TETRIS" (then start with LNK2048)
  6. Write your own BASIC program! For more info, see links above.
